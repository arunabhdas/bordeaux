## Steps

~~~
==> ng new bordeaux
~~~


~~~
==> ng generate component nav
~~~


~~~
==> ng generate component about
~~~

~~~
==> ng generate component contact
~~~

~~~
==> ng generate service data
~~~

Build for DEV
~~~
==> ng serve -o
~~~

Build for STAGING
~~~
==> ng build
~~~

Build for PROD
~~~
==> ng build --prod
~~~
